# Calculadora
Essa é uma calculadora simples feita em java usando javafx

Esse é meu primeiro usando javafx. Nesse projeto, eu estou usando JDK 8 com o nível de linguagem 8

![calculadora](https://i.imgur.com/9J5oIdi.png)

## Funcionalidades

* Soma
* Subtração
* Multiplicação
* Divisão
* Raiz Quadrada
* Potência
* Fatorial
* Porcentagem
* Inversão de Sinal
* Divisão 1 / x
